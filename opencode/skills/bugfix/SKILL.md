Use `beads` for managing tasks and subtasks.
Before starting implementation, break down the feature into smaller sub-tasks.
Refer past beads tasks for similar features to maintain consistency.

# What I do?
Reproduce the bug and verify what symptoms occur.
Check the bug report, stack trace, and any relevant logs to understand the issue.
Run the existing tests to confirm the bug and identify where it occurs.

For bug fixing, follow this
[workflow](~/.config/opencode/doc/coding_workflow.md)
